#### IDEA setting

Annotation Processor を有効にすればGradle使わなくてもcompile/testできるよ  
"Build, Execution, Deployment → Compiler → Annotation Processors"  

#### Build

```
$ ./gradlew assemble
$ java -jar build/libs/api-java-0.1-all.jar 
```
